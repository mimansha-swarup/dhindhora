## Features -

- Create a post
  - Like  
  - Comments    
- Edit & delete post
- User feed   
  - Sort by latest, oldest & trending 
- User profile
  - Add profile picture
  - Bio
  - Portfolio URL
- Follow/Unfollow
- Explore feed
- Bookmark post
- Authentication
  - Sign-up
  - Login
  - Logout
